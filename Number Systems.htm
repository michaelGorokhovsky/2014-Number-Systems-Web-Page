Number Systems

What is a Number System?

a number system is a writing system that is used to express numbers.
The number system that we most commonly use is called decimal(base 10). This system contains 10 numbers; 0 to 9. Hence base 10. Other number systems include binary(base 2) and hexadecimal(base 16). Binary only contains two numbetrs, 0 and 1. Hexadecimal contains 0 to 9 as well as A to F. A reprosents 10 B is 11 and it goes all the way to F which is 15.

What are Positional Values?

Positional values are a method of representing numbers.In the decimal system, digits to the left of the decimal point represent ones, tens, hundreds, thousands, etc. They are always multiplying by 10 and because of that, they can be expressed as powers of 10. 1 is 100, 10 is 101 100 is 102, ect. Digits to the right of the decimal point go in an opposite pattern. tenths, hundreths, and thousandths bnecome 10-1,10-2, and 10-3 respectively. An example would be the value 123.456 which breaks down into
1(102) + 2(101) 3(100) + 4(10-1) + 5(102) + 6(10-3)
In the binary number system, these values are all powers of two, so a number such as 1001.101 would break down into
1(23) + 0(22) + 0(21) + 1(20) + 1(2-1) + 0(2-2) + 1(2-3)
However in the hexadecimal system all of the posional values become powers of 16, an example would be 8A2.5D6 which breaks down into
8(162) + 10(161) + 2(160) + 5(16-1) + 14(16-2) + 6(16-3)

Counting

In any number system, counting is accomplished by advancing the rightmost or least significant digit by increments of one. When the digit being advanced has reached the last available symbol, it next returns to the first symbol and the digit to its left advances one unit.

Conversions

Binary To Decimal Conversion by Positional Values

Numbers in the binary system can be converted to decimal by adding the positional values corresponding to the digits of the binary number.
Example: 101112 => Decimal
= 1((24) + 0(23) + 1(22) + 1(21) + 1(20)
= 16 + 0 + 4 + 2 + 1
= 2310

Decimal to Binary Conversion by Remainder Method

By successively dividing a number in the decimal system by two, it can be converted into the decimal system. The remainder of each division is retained as a digit or bit of the binary number, with the first remainder as the rightmost bit.
Example: Converting 4310 into binary
43 / 2 = 21 R 1
21 / 2 = 10 R 1
10 / 2 = 5 R 0
5 / 2 = 2 R 1
2 / 2 = 1 R 0
1 / 2 = 0 R 1
Therefore 4310 = 1010112

Decimal to Hexadecimal Conversion Through Remainder Method

The remainder method that was used to convert decimal into binary can still be used to convert decimal into hexadecimal. The only difference is that instead of dividing by 2, we will now be dividing by 16. The rest of the process remains the same. Example:
527410 => Hexadecimal
5274 / 16 = 329 R 10
329 / 16 = 20 R 9
20 / 16 = 1 R 4
1 / 16 = 0 R 1
Therefore 527410 = 149A16

Hexadecimal to Decimal Conversions by Positional Values

Numbers in hexadecimal can be converted to decimal in the same way that binary is converted to decimal. To convert hexadecimal to decimal, we must simply multiply the positional values with their coresponding hexadecimal digits, and add them all together.
Example: B67F16 => Decimal
